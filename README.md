## About OverTheWire

[OverTheWire](https://overthewire.org/wargames/) is a collection of wargames designed to teach and practice computer security concepts through hands-on challenges.  
Each level builds on the previous one and introduces new topics such as Linux commands, privilege escalation, networking, cryptography, and exploit development.

## How to Use

Each level file includes:

- A short description of the **goal**  
- The **commands** used to solve it (with explanations)  
- The **password** obtained for the next level  

You can follow along by opening a terminal and connecting to the corresponding OverTheWire server.
